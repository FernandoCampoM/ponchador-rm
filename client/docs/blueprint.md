# **App Name**: TimeWise

## Core Features:

- Clock In/Out: Allow users to clock in and clock out using an API.
- Time Entry Display: Display current time entries (clock-in, clock-out times) and total hours worked, fetched from the API.
- Timecard Reporting: Provide basic reports to visualize the hours worked by users via API calls
- Clock Display: Show a big clock with the company logo on the left side of the screen.
- Keypad Input: Provide a keypad on the right side of the screen for users to enter their ID and password for clocking in/out.
- Confirmation Popup: Display an emergent popup to confirm clock-in/out action, showing the user's name, time in/out, and action type on the left side.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6), promoting a sense of reliability and precision.
- Background color: Light gray (#F0F4F7), offering a clean and unobtrusive backdrop.
- Accent color: Soft teal (#4DB6AC), providing subtle highlights for interactive elements.
- Font: 'Inter', a sans-serif font for a clean and professional look in both headings and body text.
- Use simple, consistent icons for navigation and actions, such as clock-in and clock-out.
- A clean and efficient layout emphasizing clarity and quick interaction. Use clear labels, high-contrast typography, and obvious visual hierarchy.
